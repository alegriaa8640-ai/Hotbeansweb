Useful Resources for Web Development

Below you will find a curated list of essential links for aspiring and current web developers at The Hot Beans, covering documentation, learning, and essential tools.

Official Documentation & Standards

  • MDN Web Docs - Comprehensive documentation for web technologies (HTML, CSS, JS).
  • W3C (World Wide Web Consortium) - International standards for the World Wide Web.
  • Can I Use... - Check browser support for front-end web technologies.

Learning & Training Platforms

Great resources for continuing education and skill practice.

  • freeCodeCamp - Interactive lessons and projects to learn coding.
  • The Odin Project - Full-stack curriculum focused on practical application.
  • Khan Academy - Free educational resources for various subjects, including computing.

Essential Coding Tools

  • VS Code - The most popular free source-code editor.
  • GitHub - Platform for version control and collaboration.
  • CodePen - Social development environment for front-end designers and developers.

Design and Asset Resources

  • Google Fonts - A large library of free, open-source fonts.
  • Unsplash - High-quality, free stock images.
  • Font Awesome - Vector icons and social logos.
